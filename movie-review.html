Overview

This full-stack movie review app was developed as part of a team project during an add-on course at MITS. It allows users to view and submit reviews for movies.

Tech Stack

📌 Frontend: React.js

📌 Backend: Node.js + Express

📌 Database: MongoDB

📌 Features: RESTful API, CORS integration, review handling
